# S3-CAM

**GOOUUU ESP32-S3-CAM** — ESP32-S3 + OV2640 camera module.

| App | What | Flash |
|-----|------|-------|
| [mini-games](mini-games/) | 12 games on 128×64 OLED + 5 buttons | `yarn mini-games:flash` |
| [rc-car](rc-car/) | 4WD AI follower (camera + LM Studio on PC) | `yarn rc-car:flash` |
| [folio](folio/) | Mic + camera → WiFi HTTP push | `yarn folio:flash` |

Native USB CDC: GPIO 19/20. See each app's README for wiring.
